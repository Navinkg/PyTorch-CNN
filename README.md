# 🌦️ Weather Image Classification using Deep Learning

## 📖 Overview

This project uses **Deep Learning** to classify weather conditions from images. A **pre-trained ResNet-18 Convolutional Neural Network (CNN)** is fine-tuned using **Transfer Learning** to recognize different weather categories such as sunny, cloudy, rainy, and other weather conditions. The model leverages the powerful feature extraction capabilities of ResNet-18 to achieve accurate image classification.

---

## 🎯 Objectives

- Load and preprocess weather image datasets.
- Apply data augmentation and image transformations.
- Utilize a pre-trained ResNet-18 model for transfer learning.
- Train and validate the model using PyTorch.
- Evaluate model performance on unseen weather images.
- Predict weather conditions from input images.
